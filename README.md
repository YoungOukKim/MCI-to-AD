## Abstract

We identify a statistically bounded astrocytic metabolic phase boundary that
structures the transition from compensatory resilience to accelerated vulnerability
during the MCI-to-AD continuum. Integrating SEA-AD single-cell pseudo-progression
(1.3M nuclei, 84 donors), a reversible zebrafish MCI model, murine validation,
and longitudinal ADNI CSF proteomics (n = 735), we show that astrocytic PTGDS
undergoes a biphasic trajectory — compensatory induction followed by abrupt
exhaustion — defining a discrete inflection at CPS 0.46 (Davies' p = 0.032).
Critically, astrocytic metabolic collapse precedes inflammatory amplification:
PTGDS exhaustion gates subsequent LCN2 induction and CREB/NGFR neurogenic
suppression, repositioning neuroinflammation as a downstream consequence
rather than a causal driver.
This phase boundary corresponds clinically to MMSE 26.0 (95% CI: 24.8–27.2),
defining a quantifiable pre-boundary therapeutic window.

## Reproducibility

### Requirements

- R version 4.3.2
- Key packages: `rhdf5`, `data.table`, `segmented`, `ggplot2`, `pROC`,
  `survival`, `survminer`, `patchwork`, `tidyverse`

### Installation

```r
source("environment/packages.R")
```

### Running analyses

```r
# Fig. 1 — SEA-AD trajectory analysis
# Step 1: extract expression from h5ad (requires H5AD_PATH env var)
Rscript analysis/Fig1_SEA-AD/01_SEAAD_data_extraction.R

# Step 2: generate Tables 1–3, CCF, trajectories
Rscript analysis/Fig1_SEA-AD/02_SEAAD_trajectory_analysis.R

# Step 3: generate Fig. 1 panels
Rscript analysis/Fig1_SEA-AD/05_Fig1_SEA-AD_panels.R

# Fig. 5 — ADNI proteomics (requires ADNI_PATH and SOMA_CSV env vars)
Rscript analysis/Fig5_ADNI/03_ADNI_clinical_analysis.R

# All tables (SEA-AD)
Rscript analysis/Tables/Tables_main_and_S1_S4.R

# All tables (ADNI)
Rscript analysis/Tables/TableS5_S6_ADNI_demographics_robustness.R
```

### Environment variables

```bash
export H5AD_PATH=/path/to/SEAAD_MTG_RNAseq_final-nuclei.2024-02-13.h5ad
export ADNI_PATH=/path/to/ADNIMERGE2/data/
export SOMA_CSV=/path/to/CruchagaLab_CSF_SOMAscan7k.csv
```

## Key Findings

| Finding | Value | Method |
|---------|-------|--------|
| PTGDS inflection point | CPS 0.46 | Quadratic model (β₂ = −2.07, p < 2.2e−16) |
| Segmented regression breakpoint | Bin 0.23 (95% CI: 0.13–0.33) | Davies' test (p = 0.032) |
| Clinical phase boundary | MMSE 26.0 (95% CI: 24.8–27.2) | Segmented regression, ADNI |
| Zebrafish avoidance deficit | −22.7% (p = 0.020) | Red ball avoidance, 14 dpf |
| BXP-101 cognitive rescue | +13.1 pp (p = 0.016) | BXP-101 0.6 μg/ml vs MCI |
| BLBP reactive gliosis (MCI) | +51% vs Control (p < 0.01) | Whole-mount IF, 14 dpf |
| BXP-101 gliosis suppression | −47% vs MCI (p < 0.001) | BLBP whole-mount IF |
| IL-6 suppression (protein) | −75% vs MCI | ELISA, 14 dpf |
| MCI→AD conversion risk | HR 2.89 (95% CI: 2.1–4.0) | Cox regression, ADNI |
| MCI vs AD discrimination | AUC 0.741 (95% CI: 0.698–0.784) | ROC, PTGDS/LCN2 ratio |
